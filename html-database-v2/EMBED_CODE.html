📋 Embed Code for Memorial Park Cemetery Database

Use the embed code below to integrate the cemetery database into Google Sites or any other website that supports iframe embedding.

⚠️ Important: Replace YOUR_WEBSITE_URL with the actual URL where you've hosted the database files.
Example: https://yoursite.github.io/html-database-v2/ or https://yourdomain.com/cemetery-database/

Basic Embed Code (Recommended)

This code embeds the main index page with full functionality:

<iframe src="YOUR_WEBSITE_URL/index.html" width="100%" height="800px" frameborder="0" style="border: none; border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.1);" allowfullscreen> </iframe>

Alternative: Embed Specific Pages

Search Page Only

<iframe src="YOUR_WEBSITE_URL/search.html" width="100%" height="700px" frameborder="0" style="border: none; border-radius: 8px;"> </iframe>

Map Page Only

<iframe src="YOUR_WEBSITE_URL/cemetery-map.html" width="100%" height="900px" frameborder="0" style="border: none; border-radius: 8px;"> </iframe>

📝 How to Use in Google Sites

  1. Open your Google Site in edit mode
  2. Click "Insert" in the toolbar
  3. Select "Embed" → "Embed code"
  4. Paste one of the embed codes above (replace YOUR_WEBSITE_URL first!)
  5. Click "Insert"
  6. Adjust the size if needed by clicking on the embedded content
✅ Tips:
  • Make sure all files are publicly accessible
  • Test the embed in a browser first to ensure it loads correctly
  • For mobile responsiveness, use width="100%" and adjust height as needed
  • Consider using 900px-1000px height for the map page to show the full map

🔗 Example with Real URL

If your site is hosted at https://example.com/cemetery/, your embed code would be:

<iframe src="https://example.com/cemetery/index.html" width="100%" height="800px" frameborder="0" style="border: none; border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.1);" allowfullscreen> </iframe>
📌 Remember: The database uses localStorage, so data is stored in each user's browser. For shared data across users, you'll need to implement a backend solution (Google Sheets, database, etc.).
