Paste Records from Find a Grave

Easy Method: Just copy and paste the entire list from Find a Grave. The system will automatically extract names, dates, and plot information.

Import from CSV File

Upload a CSV file with columns: Full Name, Birth Date, Death Date, Veteran Status, Branch, War/Era, Section, Plot Number, etc.

Bulk Import from JSON File

Import all 335 Find a Grave records with images: Upload a JSON file containing all records with image URLs.

JSON format: Array of objects with fields: name, birth, death, plot, imageUrl, findAGraveUrl, memorialId
How to Create JSON File:
  1. Export data from Find a Grave (or use a scraper tool)
  2. Format as JSON array with fields: name, birth, death, plot, imageUrl, findAGraveUrl
  3. Save as .json file
  4. Upload and import
