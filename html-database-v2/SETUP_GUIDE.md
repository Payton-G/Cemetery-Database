# Quick Setup Guide

## 🚀 Getting Started in 5 Minutes

### Step 1: Open the Database
1. Navigate to the `html-database` folder
2. Double-click `index.html` to open in your web browser
3. You should see the home page with navigation

### Step 2: Test Data Entry
1. Click "Data Entry" in the navigation
2. Fill in the required fields:
   - Full Name: "Test Entry"
   - Entered By: Your name
   - Entry Date: Today's date (auto-filled)
3. Click "Save Record"
4. You should see a success message

### Step 3: View Your Entry
1. Click "View Records" in the navigation
2. You should see your test entry
3. Try the search and filter features

### Step 4: Ready to Use!
The database is now ready for real data entry!

## ⚠️ Important Notes

### Data Storage
- Data is stored in browser **localStorage**
- Each browser/device has separate data
- **Regularly export data** using the "Export Data" button
- Save CSV files in the `data/` folder for backup

### Browser Requirements
- Modern browser (Chrome, Firefox, Edge, Safari)
- JavaScript must be enabled
- localStorage must be enabled

### For Multiple Users
- Current system: Each user has separate data
- **Recommendation**: Use Google Sheets integration for shared data
- See `docs/GOOGLE_SITES_SETUP.md` for details

## 🆘 Troubleshooting

### Database won't open
- Make sure you're opening `index.html` in a web browser
- Check that all files are in the correct folders
- Try a different browser

### Data not saving
- Check browser console for errors (F12)
- Ensure JavaScript is enabled
- Try clearing browser cache and reloading

### Can't see records
- Make sure you've entered at least one record
- Check the filter/search settings
- Try refreshing the page
