# User Guide - Memorial Park Cemetery Database

A simple guide for data entry users (ages 14-18).

## 🎯 Getting Started

### Opening the Database

1. Open your web browser (Chrome, Firefox, or Edge work best)
2. Navigate to the database website
3. You'll see the home page with statistics

## 📝 Entering New Records

### Step-by-Step Instructions

1. **Click "Data Entry"** in the top navigation menu

2. **Fill in Required Fields** (marked with red *)
   - **Full Name**: Type the name exactly as it appears on the marker
   - **Entered By**: Type your name or initials
   - **Entry Date**: Today's date (usually filled in automatically)

3. **Fill in Optional Fields** (as much as you can find)
   - **Birth Date** and **Death Date**: If visible on the marker
   - **Veteran Status**: Select "Yes" if the marker shows military service
     - If "Yes", you'll see more fields for Branch and War/Era
   - **Cemetery Section**: Like "Section A" or "Row 5"
   - **Plot Number**: If there's a plot number visible
   - **Marker Condition**: How the marker looks
   - **Marker Type**: What kind of marker it is
   - **Inscription**: Any text written on the marker
   - **Notes**: Anything else you notice

4. **Click "Save Record"**
   - You'll see a green success message
   - The form will clear automatically
   - You can enter another record right away

### Tips for Accurate Data Entry

✅ **DO:**
- Double-check spelling of names
- Fill in as much information as possible
- Be consistent (use "Section A" not "A" or "section A")
- Take your time - accuracy is important

❌ **DON'T:**
- Skip required fields
- Guess information you're not sure about
- Enter the same record twice
- Use abbreviations unless necessary

## 🔍 Viewing Records

### View All Records

1. Click **"View Records"** in the navigation
2. You'll see all records that have been entered
3. Use the search box to filter records
4. Use the dropdown to sort records

### Search for Specific Records

1. Click **"Search"** in the navigation
2. Type what you're looking for (name, section, etc.)
3. Use filters for Veteran Status or Marker Condition
4. Results appear as you type

## 💡 Common Questions

### Q: What if I make a mistake?
**A:** Currently, you'll need to ask an administrator to fix it. Be careful when entering data!

### Q: Can I add a photo?
**A:** Not directly in the form, but you can mention in the Notes field that a photo was taken.

### Q: What if the marker is damaged or missing?
**A:** Select the appropriate condition from the dropdown. This helps prioritize restoration work!

### Q: How do I know if someone is a veteran?
**A:** Look for military symbols, branch names, or war dates on the marker. When in doubt, select "Unknown."

### Q: What if I don't know the section or plot number?
**A:** That's okay! Fill in what you can. You can always add more information later.

## 🎓 Best Practices

### Before You Start
- Have a clear view of the marker
- Take notes on paper if needed
- Check if you have all the information

### While Entering Data
- Read the marker carefully
- Don't rush - accuracy matters
- Ask for help if something is unclear

### After Entering
- Review what you entered
- Make sure all required fields are filled
- Check for spelling errors

## 🆘 Getting Help

If you need help:
1. Check this guide first
2. Ask your supervisor or teacher
3. Look at the examples in the database
4. Contact the project administrator

## 📊 Understanding the Statistics

On the home page, you'll see:
- **Total Records**: How many records are in the database
- **Veterans**: How many veterans have been identified
- **Entries This Week**: How many records were added recently

These numbers update automatically as more data is entered!
