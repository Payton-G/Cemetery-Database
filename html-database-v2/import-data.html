Import Records from Find a Grave

Use this page to import multiple records at once. You can either:
1. Paste records from Find a Grave (one per line)
2. Import from CSV file
3. Enter records manually in bulk format

Paste Records from Find a Grave

Easy Method: Just copy and paste the entire list from Find a Grave. The system will automatically extract names, dates, and plot information.

Or use manual format: Name | Birth Date | Death Date | Plot Info (one per line)

Import from CSV File

Upload a CSV file with columns: Full Name, Birth Date, Death Date, Veteran Status, Branch, War/Era, Section, Plot Number, etc.
